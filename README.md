# Plantilla-SRS
Plantilla del Documento de Especificación de Requisitos de Software según ISO/IEC/IEEE 29148:2011
El formato markdown está basado en [jam01/SRS-Template](https://github.com/jam01/SRS-Template) que también se basa en ISO/IEC/IEEE 29148:2011 aunque es diferente, ignoramos aún el porqué.
El contenido se basa exclusivamente en las recomendaciones del estándar, no se usó otras fuentes aún.
